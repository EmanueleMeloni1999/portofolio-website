# Emanuele Meloni - Portfolio

Portfolio professionale per **3D Rigger | Pipeline TD | Technical Animator**

## Features

- Design responsive (mobile, tablet, desktop)
- Animazioni CSS custom e intro sprite animation
- ShowReel YouTube embed
- Contact form (Web3Forms)
- SEO-friendly

## Tech Stack

- HTML5, CSS3, JavaScript (Vanilla)
- Responsive design con CSS Grid/Flexbox
- GitHub Pages / Static hosting

## Local Development

```bash
git clone https://github.com/EmanueleMeloni1999/portofolio-website.git
cd portofolio-website
python -m http.server 8000
```

## Project Structure

```
portfolio/
├── assets/

├── data/

│   ├── all-personal-work.json  # Progetti personali

│   ├── skills.json      # Skills e software

│   └── interactions.js  # Tiger head mouse tracking

│   └── main.css         # Tutti gli stili
├── projects/            # Pagine dettaglio progetti
├── index.html           # Homepage

```
